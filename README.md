# Federated Learning for Medical Imaging

## Overview
This project implements a Federated Learning framework for medical image analysis using deep learning techniques.
The system enables distributed model training while preserving data privacy across multiple medical datasets.



## Features
- Federated Learning Architecture
- Medical Image Processing
- Flask Dashboard Integration
- Deep Learning Model Training
- Distributed Client-Server Communication
- Performance Visualization

## Technologies Used
- Python
- TensorFlow
- Flask
- NumPy
- OpenCV
- PyTorch
